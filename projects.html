Portfólio com Assistente de IA Integrado

Portfólio desenvolvido em HTML e CSS com layout responsivo que apresenta trajetória profissional de forma interativa. Conta com assistente de IA conversacional integrado que responde perguntas sobre carreira, projetos e habilidades, funcionando como guia inteligente do site.

Tecnologias: HTML, CSS, JavaScript, Python, Flask, Google Gemini, LangChain, FAISS.

Pod&Dev - Player de Podcast com IA

Desenvolvi uma interface web para apresentar o podcast Pod&Dev, contornando a limitação do GitHub de não suportar players de áudio. O player permite a audição do episódio, que foi roteirizado, narrado e ilustrado com capa gerada por IA Generativa.

Tecnologias: HTML, CSS, Microsoft Clipchamp, IA Generativa (Gemini 2.5 Flash, ElevenLabs).
